# NL-UI / Notlimey UI

Is a library for ui components

# Getting started

`npm i --save nl-ui`

## Example

```javascript
import './App.css';
import { TitleHelmet} from 'nl-ui'

function App() {
  return (
    <div className="App">
      <TitleHelmet Title="Hey" SubTitle="cool" />
    </div>
  );
}

export default App;```
